1.1.3、httpd的常见选项

语法 : httpd [选项]

选项 作用
-l 查看编译模块
-M 输出一个已经启用的模块
-V 显示版本
-X 以调试模式运行httpd
-t 检查配置文件是否有语法错误

1.2、curl命令

curl是基于url语法在命令行方式下工作的文件传输工具，它支持FTP,FTPS,等协议。

curl支持以下功能：

  • https认证
  • http的POST/PUT等方法
  • ftp上传
  • 代理服务器
  • cookies
  • 用户名/密码认证
  • 下载文件断电续传
  • sockes5代理服务
  • 通过http代理服务器上传到ftp

语法：curl [选项] [url]

常用选项：

-A 设置代理发送服务器

-basic 使用http基本认证

–tcp-nodelay 使用TCP_NODRLAY选项

-o 输出写到文件中

-I 只显示报文信息

–limit-rate 设置传输速度

权限 作用
Require all granted 允许所有主机访问
Require all deny 拒绝所有主机访问
Require ip IPADDR 允许指定ip访问
Require no ip IPADDR 不允许指定ip访问
Require host HOSTNAME 指定主机名访问
Require no host HOSTNAME 不允许指定的主机名访问
