Mysql查询操作

2022/10/13 数据库 linux mysql

1、普通查询

select * from t_user;

条件查询

关键字where

符号 说明
AND 与
OR|IN(OBJECT,….) 或

查询id=1的


select * from t_user where id=1;

查询age=18且username=lzj

select * from t_user where age=18 and username=lzj;

查询username为lisi,zhangsan

select * from t_user where username in (lisi,zhangsan)

排序

关键字 说明
desc 降序
asc 升序 
select * from t_user order by id desc;

模糊查询

SELECT * from users WHERE username LIKE '张%';

4、联表查询

select * from users,user_role,role where users.id=user_role.uid and role.id=user_role.id

5、左右连接查询
