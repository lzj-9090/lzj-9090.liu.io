Mysql的用户和权限

2022/10/11 数据库 linux mysql

mysql密码修改

修改方式有两种：

  • 通过root用户修改
  • 通过修改配置文件的方式修改

1、通过root来修改密码

set password for 'lzj'@'localhost'="密码"

2、修改配置文件的方式

1.停止mysqld服务

    systemctl stop mysql

2.在/etc/mysql/mysql.conf.d/mysqld.cnf最后加上

skip-grant-tables

3.启动mysqld服务

   systemctl start mysql

4.登录mysql服务器

​ mysql (免密码登录)

5.查看相关的数据，root的加密方式为

mysql_native_password

6.修改密码为空

update mysql.user set authentication_string='' where user='root';

\7. 去掉步骤2中的skip-grant-tables

8.再次重启mysql服务

systemctl restart mysql

9.再次修改密码

   mysqladmin -u root -p password “123456”

其中旧密码为空
