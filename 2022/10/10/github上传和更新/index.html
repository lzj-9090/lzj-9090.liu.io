Github上传操作

2022/10/10 开发 git linux

1、创建账号

这个就不细说了

2、建立本地和github账号的联系

在git上设置自己github用户名、邮箱、密码

git config --global user.email "github 邮箱"
git config --global user.password "github 密码"
ssh git@github.com
git config --global user.name "github 名称"

生成密钥

ssh-keygen -t rsa -C "github 邮箱"

复制公钥在github中

测试连接

ssh -T git@github.com

3、上传文件

初始化

git init

查看状态

git status

添加上传的目录

#全部上传
git add .

添加上传内容介绍

 git commit -m "hexo"

上传

git push -u origin master
