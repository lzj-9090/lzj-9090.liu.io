2022/10/24 数据库 linux mysql

mysql的二进制安装方式

安装

下载地址：

依赖：gcc、libtinfo.so.5

1、创建mysql用户和组

groupadd -r mysql
useradd -M -s /sbin/nologin -g mysql mysql

2、解压二进制安装包

tar -xf mysql-8.0.31-linux-glibc2.12-x86_64.gz -C /usr/local

3、创建软连接

ln -sv mysql-8.0.31-linux-glibc2.12-x86_64.gz /usr/local/mysql

4、创建数据存储地址

mkdir -p /opt/mysql/data

5、赋予权限

chown -R mysql.mysql /usr/local/mysql
chown -R mysql.mysql /opt/mysql

6、初始化配置文件

 mysqld --initialize --user=mysql --datadir=/opt/mysql/data

7、编写配置文件

 cat /etc/my.cnf
[mysqld]
basedir = /usr/local/mysql
datadir = /opt/mysql/data
socket = /tmp/mysql.sock
port = 3306
pid-file = /opt/mysql/data/mysql.pid
user = mysql
skip-name-resolve
#skip-grant-tables			用于免密登录

8、将mysql写入Linux自动管理

cp -a /usr/local/mysql/support-files/mysql.server /etc/init.d/mysqld
sed -ri 's#^(basedir).*#\1/usr/local/mysql#g' /etc/init.d/mysqld
sed -ri 's#^(datadir).*#\1/opt/mysql/data#g' /etc/init.d/mysqld

9、启动mysql

service mysql start

10、登入修改密码

mysql -uroot -p"初始密码"

11、修改密码

update user set authentication_string='' where user='root';
flush privileges;
alter user 'root'@'localhost' identified with mysql_native_password by 'rt1234';

问题解决

1、在初始化的时候报错

解决：删除关于mysql配置文件my.cnf、data下面的文件

2、mysql修改完密码还是不能登录

解决：建议重启
